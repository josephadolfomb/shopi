# Shopi

This project is an E-Commerce Website developed using React, Vite & Tailwind CSS.

## Skills

React, Vite, TailWindCSS, Context, HeroIcons & Netlify Deploy.

## Features

- Search bar
- Filtered products by category and title
- Add to cart
- Checkout
- Orders
- Shopping Cart count
